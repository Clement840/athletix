# 🏃 Athle Tracker

## 📖 Description

Athle Tracker est une application web permettant aux athlètes de suivre leurs entraînements, leurs compétitions et leur progression au fil du temps.

Contrairement à une application spécialisée dans la course à pied, Athle Tracker est conçu pour s'adapter à l'ensemble des disciplines de l'athlétisme :

- Sprint
- Demi-fond
- Fond
- Haies
- Sauts
- Lancers
- Épreuves combinées
- Marche athlétique

L'objectif est de proposer un carnet d'entraînement intelligent, un suivi des performances en compétition et des statistiques adaptées à chaque discipline.

---

## 🚀 Fonctionnalités prévues

- Gestion des profils athlètes
- Carnet d'entraînement
- Historique des compétitions
- Suivi des records personnels
- Tableaux de bord
- Graphiques de progression
- Statistiques personnalisées selon la discipline
- Comparaison entre plusieurs athlètes
- Objectifs de saison
- Import automatique des résultats FFA (si possible)
- Export des données
