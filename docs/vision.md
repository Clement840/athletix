# Vision du projet

## Pourquoi ce projet ?

Athle Tracker est né d'un besoin personnel : disposer d'une application permettant de suivre facilement les entraînements, les compétitions et la progression en athlétisme.

Les applications existantes sont souvent orientées vers la course à pied ou ne répondent pas aux besoins spécifiques des différentes disciplines de l'athlétisme.

L'objectif est donc de développer une application moderne, intuitive et évolutive, adaptée à tous les athlètes.

---

## Public visé

L'application est destinée principalement :

- aux athlètes ;
- aux entraîneurs ;
- aux clubs d'athlétisme.

À terme, plusieurs utilisateurs pourront partager leurs données et comparer leurs performances.

---

## Principes de conception

### Une seule application pour toutes les disciplines

L'application ne doit pas être pensée uniquement pour les coureurs.

Elle doit fonctionner aussi bien pour :

- les sprinteurs ;
- les hurdlers ;
- les sauteurs ;
- les lanceurs ;
- les marcheurs ;
- les athlètes des épreuves combinées.

L'interface et les statistiques devront s'adapter automatiquement aux disciplines pratiquées.

---

### Une application centrée sur les données

Toutes les informations (entraînements, compétitions, records...) seront enregistrées de manière structurée afin de permettre des analyses avancées.

Les graphiques et tableaux de bord seront générés automatiquement à partir de ces données.

---

### Une interface simple

L'application doit rester agréable à utiliser.

L'ajout d'une séance ou d'une compétition doit être rapide, avec le moins de saisie possible.

---

### Une architecture évolutive

Le projet est conçu pour pouvoir accueillir facilement de nouvelles fonctionnalités :

- import automatique des résultats FFA (si possible) ;
- synchronisation avec des montres GPS ;
- comparaison entre athlètes ;
- statistiques avancées ;
- intelligence artificielle.

---

## Objectif à long terme

Créer une application suffisamment complète pour être utilisée au quotidien par un athlète ou un entraîneur, tout en restant simple à maintenir et à faire évoluer.
